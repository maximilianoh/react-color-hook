SketchPresetColors with custom titles
```js
import SketchPresetColors from './SketchPresetColors.jsx';
const colors = [
    {
      color: '#fff',
      title: 'white',
    },
    {
      color: '#999',
      title: 'gray',
    },
    {
      color: '#000',
    },
    '#f00',
  ];

<SketchPresetColors colors={colors} />
```
